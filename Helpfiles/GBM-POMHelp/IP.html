Connecting to the Lenz 23151 LAN Interface(s)

This program is designed to communicate with Lenz 23151 LAN Interfaces. Other types of interfaces (such as USB interfaces) are not supported.

For sending PoM-write messages, this program establishes a TCP connection to a Lenz 23151 LAN interface.

For receiving CV-values, this program can connect to the same interface as for sending PoM-write messages (for that purpose it will establish a second TCP connection). It can also listen to a second Lenz 23151 LAN interface, however.

The IP address(es) can be entered via the preferences window.
