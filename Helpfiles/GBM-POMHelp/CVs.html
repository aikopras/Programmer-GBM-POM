0x00110000 - Track Occupancy decoder

Basic Track Occupancy (Feedback) decoder








0x00110001 - Track Occupancy decoder with reverser

Reverser on extension board








0x00110010 - Track Occupancy decoder with relays

Four relays on extension board








0x00110100 - Track Occupancy decoder with speed measurement

Speed is displayed on connected LCD








0x01000000 - Function Decoder









0x10000000 - Watchdog and safety decoder

29


Config

R

128



Accessory Decoder configuration

This CV is checked by the software within the decoder








Bit 7: ‘0’ = Multi-function (Loco) Decoder / ‘1’= Accessory Decoder

All my decoders should be Accessory








Bit 6: Addressing Method: ‘0’= Decoder Address; ‘1’ = Output Address

All my decoders should do Decoder Addressing (=9 bit) 








Bit 5: Type: ‘0’ = Basic Accessory; ‘1’ = Extended Accessory

All my decoders should be Basic Accessory








Bit 4 = Reserved for future use.









Bit 3 = BiDi: "0" = no support, "1" = support

None of my decoders support the BiDi standard








Bit 0..2 = Reserved for future use.
